# Contributing to cj-simulator

Thanks for your interest in contributing!

## Development Setup

1. Clone the repo: `git clone https://github.com/NathanNorman/cj-simulator.git`
2. Open `index.html` in your browser to play the game
3. No build process required -- just edit and refresh

## Pull Request Process

1. Fork the repo and create your branch from `main`
2. Add tests for any new functionality
3. Ensure all tests pass
4. Update documentation if needed
5. Submit a pull request

## Code Style

- Follow existing code conventions
- Keep changes focused and atomic

## What We Accept

- Bug fixes
- Performance improvements
- Documentation improvements
- New features (please open an issue first to discuss)
